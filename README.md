## Installation

1. **Clone the repository**

```bash
git clone https://github.com/INSAG40/frontend.git
cd frontend
```

2. **Install dependencies**

```bash
npm install
```

---

## Running the Project

Start the development server:

```bash
npm run dev
```

Open the **Local** URL in your browser (usually `http://localhost:5173/`) to see the project in action.

---

## Checking the Project

1. **Linting**

```bash
npm run lint
```

Checks your TypeScript code and formatting.

2. **Build for production**

```bash
npm run build
```

Generates production-ready files in the `dist/` folder.

3. **Preview production build**

```bash
npm run preview
```

Runs a local server to preview the built project.

## Project Structure

```
frontend/
├─ src/                # Source files (components, pages, assets)
├─ backend/            # Backend-related files (if any)
├─ index.html          # Main HTML file
├─ package.json        # Project dependencies & scripts
├─ tailwind.config.js  # Tailwind CSS configuration
├─ vite.config.ts      # Vite configuration
└─ tsconfig.json       # TypeScript configuration
```
